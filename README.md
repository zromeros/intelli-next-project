## How to install

1. Clone repository:

2. Install:


    - cd server
    - npm install
    - cd ../client
    - npm install
    - cd ..

3. Run Backend:


    - cd server
    - npm run dev

4. Run Frontend:


    - open new terminal in project folder
    - cd client
    - npm start
